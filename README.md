# Weather App

A simple and elegant web application that displays current weather information for any city worldwide. 
It uses the OpenWeatherMap API to fetch real-time data and presents temperature, humidity, wind speed, and weather icons in a clean interface.

# Features

- Search weather by city name
- Display temperature in Celsius
- Show humidity percentage
- Display wind speed in km/h
- Dynamic weather icons based on conditions
- Error handling for invalid city names

## Technologies Used

- **HTML5** – Structure and layout
- **CSS3** – Styling and responsive design
- **JavaScript** – Logic and API integration
- **OpenWeatherMap API** – Weather data source
